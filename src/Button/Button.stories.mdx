## Props

| Prop Name   | Type                                      | Default                  | Required | Description                           |
| :---------- | :---------------------------------------- | :----------------------- | :------: | :------------------------------------ |
| variant     | <code>"primary" &#124; "secondary"</code> | <code>"secondary"</code> |    -     | Button's appearance                   |
| destructive | boolean                                   | false                    |    -     | If button's action is destructive     |
| disabled    | boolean                                   | false                    |    -     | If button is disabled                 |
| fullWidth | boolean                                |    -    |    -     | Button takes up 100% of its container |
| onClick     | function                                  | -                        |    -     | Callback for Button's `onClick` event |

</div>
